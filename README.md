Building A REST API For Instagram With Spring Boot, Spring Security , PostgreSQL and JWT

-------------------------------
Tools Used -
1. Spring Boot 3.0.9
2. PostgreSQL
3. Lombok
4. JSON Web Token (JWT) by JJWT
5. Maven
6. Spring Data JPA
7. Spring Security
8. SpringDoc OpenAPI
9. Docker
